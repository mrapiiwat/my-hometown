The Problem

LLM ทั่วไปไม่รู้จัก "ท่าอิฐ"

The Brain (Python)

RAG Engine Class

หัวใจหลักของระบบที่ทำหน้าที่:

  • โหลด Index ข้อมูลท้องถิ่น (FAISS)
  • ค้นหาเอกสารที่เกี่ยวข้อง (Similarity Search)
  • เชื่อมต่อกับ LLM เพื่อสร้างคำตอบ

The Body (API)

Flask Route

จุดเชื่อมต่อระหว่าง iOS และ Python:

  • รับ JSON Request จาก App
  • ส่งข้อความเข้า RAG Engine
  • ส่งคืน JSON คำตอบกลับไปแสดงผล
@app.route('/ask', methods=['POST']) def ask_bot(): data = request.json user_msg = data['message'] # Process with RAG answer = rag.ask(user_msg) return jsonify({ "reply": answer, "status": "success" })
